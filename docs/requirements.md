# 需求规格说明书

## 项目名称
生活管理 · Life Manager — 私人生活管理助手 PWA

## 核心功能模块

### 一、日常记录模块（AI 对话 + 摘录 + 日记）
1. **AI 对话交互** — 输入框直接与 DeepSeek AI 对话
2. **三种类型自动分流**：
   - `schedule` 日程类：含时间/待办/计划/打卡 → 自动预填日程表单，可绑定目标
   - `chat` 闲聊类：仅展示 AI 回复，不自动生成日程
   - `note` 随笔文学类：自动归档 + AI 点评 + 书籍名句
3. **自由摘录功能**：
   - 移动端：长按框选文本 → 弹出【摘录】按钮
   - 桌面端：选中文字后 Enter 键触发摘录
   - 不限次数摘录 → 临时缓存区
   - 【完成摘录】→【AI 一键总结】→ 生成文艺日记卡片
4. **日记卡片**：绑当日日期 → 存入本地日记库 → 可在日历视图查看
5. **自动归类**：6大分类（学习/事业/物质资产/健康/情感人际/娱乐心愿）
6. **AI 配置面板**：API 密钥（掩码显示）+ 自定义接口地址 + 重置默认

### 二、人生目标模块
1. **预设目标**（6大分类各有预设）
2. **可折叠 Markdown 编辑器** — 主目标下方无限新增分支计划
3. **AI 一键总结建议** — 汇总所有目标输出阶段性行动建议

### 三、日程模块
1. 快捷日期按钮：今日、明日、本周、下周
2. 日历视图 + 列表视图
3. 日历单元格增加日记标识，点击日期弹窗展示日记卡片
4. 微信读书文艺卡片风格统一

### 四、设置页
1. AI API 密钥 + 接口地址配置
2. 数据导出为 JSON 文件下载
3. 数据导入从 JSON 文件恢复
4. 操作结果提示

### 五、硬性约束
- 纯前端，无后端，无第三方依赖
- LocalStorage 本地存储
- 3 栏底部导航：日程、人生目标、日常记录
- 微信读书文艺风格 UI
- 增量修改，不改动核心联动逻辑
