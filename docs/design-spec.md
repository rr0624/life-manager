# UI 设计规范 — 微信读书文艺风格

## 设计理念
参考微信读书 App 的文艺阅读风格：大面积留白、柔和弥散浅阴影、衬线标题字体、浅米低饱和配色、轻薄圆角卡片、宽松阅读行间距。

## 色彩系统
```
背景主色:  #f7f4ef  (暖白/米色)
卡片白:    #ffffff
输入区:    #f0ede6
文字主色:  #2c2c2c
文字次色:  #8c8c8c
文字弱色:  #b0b0b0
强调色:    #c9a87c  (暖金色，用于按钮/链接)
成功色:    #7a9a7a  (低饱和绿)
危险色:    #c08080  (低饱和红)
```

## 六大分类色
```
学习:     #b8a9c9  (淡紫)
事业:     #8aadc9  (淡蓝)
物质资产: #c9b896  (淡金)
健康:     #a3c9b5  (淡绿)
娱乐心愿: #c9a3b5  (淡粉)
情感人际: #c9b5a3  (淡棕)
```

## 字体
```css
标题: 'Noto Serif SC', 'Source Han Serif SC', 'Songti SC', Georgia, serif
正文: -apple-system, 'PingFang SC', 'Microsoft YaHei', sans-serif
```

## 间距规范
- 页面左右边距: 20px
- 卡片内边距: 20px
- 卡片间距: 12px
- 段落行高: 1.8
- 标题行高: 1.4
- 阅读字体大小: 15px
- 辅助字体大小: 12px

## 圆角
- 小圆角 (标签/芯片): 8px
- 卡片: 12px
- 大卡片/模态: 16px
- 头像/图标按钮: 50%

## 阴影
```css
/* 极淡阴影 — 营造"浮在纸面上"的感觉 */
--shadow-card: 0 2px 16px rgba(0,0,0,0.04);
--shadow-modal: 0 8px 40px rgba(0,0,0,0.08);
```

## 组件规范

### 卡片
- 白色背景 + 极淡阴影
- 12px 圆角
- 20px 内边距
- 宽松行间距

### 输入框
- 浅米背景 `#f0ede6`
- 无边框，聚焦时底部出现暖金色下划线
- placeholder 用文字弱色

### 按钮
- 暖金色背景 `#c9a87c`，白色文字
- 12px 圆角
- 轻压感（active 缩放）

### 导航栏
- 白色毛玻璃
- 3 个等宽 Tab
- 选中态暖金色图标+文字

### 聊天气泡
- 用户: 暖金色，靠右
- AI: 浅灰，靠左
- 宽松圆角，形似微信气泡
